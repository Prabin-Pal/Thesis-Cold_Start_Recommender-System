# Hybrid Cold Start Recommender System

This project presents a hybrid recommendation system designed to address the cold-start problem using the Amazon Beauty dataset (5-core subset).

## Features
- Data preprocessing and cleaning
- Hybrid recommendation approach
- Evaluation of model performance

## Dataset
Amazon Beauty Dataset (5-core):
https://nijianmo.github.io/amazon/index.html

## Source Code
This repository contains all implementation files used in the thesis.
